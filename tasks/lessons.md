# Lessons

- 2026-04-20：当用户明确说明运行环境是 Colab + GPU 时，不要把 Whisper 设计成默认本地依赖；应优先把转录依赖限定在 Colab 运行时，并把本地代码保持为可选开发环境。
- 2026-04-21：用户已经验证过可工作的 yt-dlp 命令时，应优先对齐该命令的格式选择和后处理参数；不要自定义强制视频格式，尤其是 Twitter/X 这类站点可能选中大文件 HTTPS 直连导致超时。
- 2026-04-21：yt-dlp 通用下载器必须同时保留 mp4 视频和 mp3 音频能力；视频模式遇到 Twitter/X 这类站点时，应优先 HLS，并把 HTTP 兜底限制在低码率/低分辨率，避免再次选择 `http-2176`。
- 2026-04-21：通用链接的转录音频应优先使用 yt-dlp 直接提取的 mp3；不要为了复用最终 mp4 而从合并视频中再抽音频，否则可能降低 Whisper 转录准确度。
